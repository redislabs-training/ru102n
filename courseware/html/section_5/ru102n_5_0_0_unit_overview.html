Unit Overview

Welcome to Unit 5 of RU102N. In this unit, we'll be covering the last two primary use cases of Redis Stack:

  • Storage and Retrieval of serialized JSON documents
  • Indexing an Querying JSON documents stored in Redis

We'll be learning how to do all of this using Redis OM .NET, an Object Mapping library that facilities the storage and retrieval of your classes in Redis using LINQ.
